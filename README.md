# hello

This is simple application for testing the GitHub Actions infrastructure.

## Usage

The webserver listens on 0.0.0.0:8080 and responses with "`Hello world!`".

## How to build

    $ git clone https://git.dvb.digicable.hu/benedekl/hello.git
    $ cd hello
    $ mkdir build
    $ cd build
    $ cmake ../
    $ sudo make install
